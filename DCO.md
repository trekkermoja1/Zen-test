## Sign-off Procedure

All commits must include a `Signed-off-by:` line to indicate that the contributor
accepts the DCO. This can be done automatically using the `-s` flag with git:

```bash
git commit -s -m "Your commit message"
```

This will add a line like:

```
Signed-off-by: Your Name <your.email@example.com>
```

## Verification

All pull requests are checked for DCO sign-off. Commits without proper
sign-off will not be accepted until corrected.
