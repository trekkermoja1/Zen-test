## Quick Reference

### Network Scanning
```bash
nmap -sV target.com                    # Service detection
masscan 0.0.0.0/0 -p80,443 --rate 1000 # High-speed scan
scapy                                  # Python packet crafting
tshark -r capture.pcap                 # Analyze PCAP
```

### Web Testing
```bash
burpsuite                              # Start Burp
sqlmap -u "http://target.com?id=1"     # SQL Injection test
nikto -h target.com                    # Web scan
ffuf -u http://target.com/FUZZ -w wordlist.txt  # Fuzzing
gobuster dir -u http://target.com -w wordlist   # Directory scan
```

### AD Pentesting
```bash
netexec smb 192.168.1.0/24             # SMB enumeration
responder -I eth0                      # Poisoning
bloodhound-python -d domain.local -u user -p pass  # AD data
eum4linux -a target                    # SMB enum
```

### Code Analysis
```bash
semgrep --config=auto .                # Static analysis
trufflehog filesystem .                # Secret scanning
gitleaks detect -v .                   # Git leaks
bandit -r .                            # Python security
```
