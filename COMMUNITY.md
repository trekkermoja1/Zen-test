## 🤝 Contributing

### Ways to Contribute
1. **Code**: Submit PRs for features or bug fixes
2. **Documentation**: Improve docs, write tutorials
3. **Testing**: Test new features, report bugs
4. **Community**: Help others in Discord/Discussions
5. **Content**: Write blog posts, create videos

### Good First Issues
Look for issues labeled:
- `good first issue`
- `help wanted`
- `documentation`

### Development Setup
```bash
git clone https://github.com/SHAdd0WTAka/zen-ai-pentest.git
cd zen-ai-pentest
python -m venv venv
source venv/bin/activate
pip install -e ".[dev]"
```

## 📅 Community Events

### Monthly Community Call
- **When**: First Tuesday of each month, 6 PM CET
- **Where**: Discord voice channel
- **Agenda**: Roadmap updates, Q&A, showcase

### CTF Events
We organize quarterly CTF events:
- **Spring CTF**: March
- **Summer CTF**: June
- **Fall CTF**: September
- **Winter CTF**: December

### Conference Participation
- Black Hat Arsenal (submitted)
- DEF CON AI Village (planned)
- Local security meetups

### Hall of Fame
Top contributors are recognized in our [CONTRIBUTORS.md](CONTRIBUTORS.md).

### Badges
Earn badges for contributions:
- 🥇 **First Commit**: First merged PR
- 🐛 **Bug Hunter**: Found and reported 5+ bugs
