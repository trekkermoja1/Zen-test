## 🤝 How to Contribute

We welcome contributions of all kinds! Whether you're a developer, security researcher, writer, or enthusiast, there's a place for you in our community.

### Ways to Contribute

| Contribution Type | Description | Skill Level |
|-------------------|-------------|-------------|
| 💻 **Code** | Fix bugs, implement features, improve performance | Intermediate+ |
| 📖 **Documentation** | Write guides, improve README, fix typos | Beginner+ |
| 🐛 **Bug Reports** | Report issues with detailed reproduction steps | All levels |
| 💡 **Feature Ideas** | Suggest new features or improvements | All levels |
| 🧪 **Testing** | Test new releases, verify bug fixes | All levels |
| 🎨 **Design** | UI/UX improvements, logos, graphics | All levels |
| 📣 **Promotion** | Share on social media, write blog posts | All levels |
| 🌍 **Translation** | Translate docs to other languages | All levels |
| 👥 **Community Help** | Answer questions, mentor newcomers | All levels |

### Quick Start for Contributors

1. **Fork the Repository**
   ```bash
   git clone https://github.com/SHAdd0WTAka/zen-ai-pentest.git
   cd zen-ai-pentest
   ```

2. **Set Up Development Environment**
   ```bash
   python -m venv venv
   source venv/bin/activate  # Windows: venv\Scripts\activate
   pip install -e ".[dev]"
   ```

3. **Create a Branch**
   ```bash
   git checkout -b feature/your-feature-name
   ```

4. **Make Your Changes**
   - Follow our coding standards
   - Add tests for new features
   - Update documentation as needed

5. **Submit a Pull Request**
   - Fill out the PR template completely
   - Link related issues
   - Request review from maintainers

### First-Time Contributors

Look for issues labeled:
- 🏷️ `good first issue` - Perfect for newcomers
- 🏷️ `help wanted` - Community assistance needed
- 🏷️ `documentation` - Documentation improvements

> 💡 **Tip:** Don't hesitate to ask questions! We're here to help.

## 📜 Code of Conduct Reference

We are committed to providing a welcoming and inclusive experience for everyone.

### Our Values

1. **Be Respectful** - Treat everyone with respect. Healthy debate is encouraged, but harassment is not tolerated.
2. **Be Inclusive** - Welcome newcomers and help them get started.
3. **Be Constructive** - Provide constructive feedback and accept it gracefully.
4. **Be Professional** - Maintain professionalism in all interactions.
5. **Be Responsible** - Follow responsible disclosure for security issues.

### Quick Reference

- ✅ **DO:** Use welcoming and inclusive language
- ✅ **DO:** Respect differing viewpoints and experiences
- ✅ **DO:** Accept constructive criticism gracefully
- ✅ **DO:** Focus on what's best for the community
- ✅ **DO:** Show empathy towards others

- ❌ **DON'T:** Use derogatory language or personal attacks
- ❌ **DON'T:** Engage in harassment or trolling
- ❌ **DON'T:** Share others' private information
- ❌ **DON'T:** Post spam or unsolicited promotional content

### Security-Specific Guidelines

When discussing security:
- 🔒 Never share real pentest data or client information
- 🔒 Follow responsible disclosure for vulnerabilities
- 🔒 Don't use Zen AI Pentest on systems you don't own without permission
- 🔒 Report security issues privately to security@zen-ai-pentest.dev

### Reporting Issues

If you experience or witness unacceptable behavior:

1. **Immediate Concern:** Contact any moderator on Discord
2. **Code of Conduct Violation:** Email conduct@zen-ai-pentest.dev
3. **Security Issue:** Email security@zen-ai-pentest.dev

All reports are handled confidentially.

**Full Code of Conduct:** [CODE_OF_CONDUCT.md](../CODE_OF_CONDUCT.md)
