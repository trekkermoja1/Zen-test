# OpenSSF Best Practices - Setup Guide

## 🎯 Ziel: Neues Projekt für Zen-AI-Pentest registrieren

### Schritt 1: Projekt erstellen

1. **Gehe zu**: https://www.bestpractices.dev/en

2. **Klicke**: Grünen Button "**Get Your Badge Now**" oder "**Add New Project**"

3. **Wähle**: "**GitHub**" als Repository-Typ

4. **Repository auswählen**:
   - Suche nach: `SHAdd0WTAka/Zen-Ai-Pentest`
   - Oder kopiere die URL: `https://github.com/SHAdd0WTAka/Zen-Ai-Pentest`

5. **Klicke**: "**Submit**" oder "**Create Project**"

6. **Warte**: Das System scannt das Repository (dauert 1-2 Minuten)

### Schritt 2: Badge URL kopieren

Nach dem Erstellen siehst du:
- **Projekt ID**: (z.B. 12345 - merke dir diese Nummer!)
- **Badge URL**: `https://www.bestpractices.dev/projects/[ID]/badge`
- **Markdown**: `[![OpenSSF](https://www.bestpractices.dev/projects/[ID]/badge)](https://www.bestpractices.dev/projects/[ID])`

**Kopiere die Projekt ID!**

### Schritt 3: Aktuelle Status prüfen

Auf der Projektseite siehst du:
- Aktueller Score (sollte höher als 10% sein mit unseren neuen Dokumenten)
- Fehlende Kriterien
- Verbesserungsvorschläge

### Schritt 4: Sofort-Scan triggern

Um den Scan sofort zu starten:
1. Scrolle auf der Projektseite nach unten
2. Suche nach: "**Edit**" oder "**Update**" Button
3. Klicke "**Submit**" (ohne Änderungen)
4. Dies erzwingt einen sofortigen Re-Scan
