Schnellstart und Installationsanleitung

Schnellstart

In 3 Schritten starten

Beginnen Sie in wenigen Minuten mit Zen-AI-Pentest. Folgen Sie diesen einfachen Schritten.

1

Repository klonen

Klonen Sie das Repository von GitHub

git clone https://github.com/SHAdd0WTAka/Zen-Ai-Pentest.git
2

Umgebung einrichten

Wechseln Sie ins Verzeichnis und kopieren Sie die .env Datei

cd Zen-Ai-Pentest && cp .env.example .env
3

Container starten

Starten Sie die Anwendung mit Docker Compose

docker-compose up -d
Installationsmethoden

Wählen Sie Ihre Installationsmethode

Zen-AI-Pentest unterstützt verschiedene Installationsmethoden für maximale Flexibilität.

Empfohlen

Docker

Die einfachste Methode mit Docker Compose

docker-compose up -d
  • Automatische Abhängigkeiten
  • Isolierte Umgebung
  • Einfaches Upgrade

Manuell

Installation mit pip und virtueller Umgebung

pip install -r requirements.txt
  • Volle Kontrolle
  • Entwicklungsfreundlich
  • Anpassbar

Dev Container

VS Code mit Dev Containers Extension

code . && F1 > Reopen in Container
  • Konsistente Umgebung
  • VS Code Integration
  • Keine lokale Installation
Systemanforderungen

Systemvoraussetzungen

Stellen Sie sicher, dass Ihr System die folgenden Mindestanforderungen erfüllt.

Docker

20.10+

Docker Compose

2.0+

RAM

4GB Minimum

Speicherplatz

10GB

Empfohlene Spezifikationen

Für optimale Performance empfehlen wir 8GB RAM und SSD-Speicher. Die Anforderungen können je nach Scan-Umfang und gleichzeitigen Prozessen variieren.

Konfiguration

Umgebungsvariablen

Konfigurieren Sie die Anwendung über die .env Datei. Diese Variablen sind für den Betrieb erforderlich.

.env Konfiguration

Kopieren Sie .env.example nach .env

DATABASE_URL Erforderlich

PostgreSQL Verbindungs-URL

DATABASE_URL=postgresql://user:pass@localhost:5432/zenpentest
REDIS_URL Erforderlich

Redis Verbindungs-URL für Caching und Queues

REDIS_URL=redis://localhost:6379/0
OPENAI_API_KEY Optional

OpenAI API Key für KI-gestützte Analysen

OPENAI_API_KEY=sk-...
JWT_SECRET Erforderlich

Geheimer Schlüssel für JWT Token-Generierung

JWT_SECRET=your-super-secret-jwt-key-min-32-chars
